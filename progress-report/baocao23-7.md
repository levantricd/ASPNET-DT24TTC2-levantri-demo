# Báo cáo tiến độ 02

## Thời gian
23/07/2026

## Nội dung đã hoàn thành

- Thiết kế mô hình cơ sở dữ liệu của hệ thống.
- Tạo các lớp Model:
  - User
  - Specialty
  - Doctor
  - Schedule
  - Appointment
- Tạo lớp ApplicationDbContext.
- Khai báo các DbSet cho Entity Framework Core.
- Cấu hình chuỗi kết nối SQL Server trong appsettings.json.
- Đăng ký DbContext trong Program.cs.
- Build dự án thành công.

## Kế hoạch tiếp theo

- Tạo Migration.
- Khởi tạo cơ sở dữ liệu trên SQL Server.
- Kiểm tra các bảng được tạo.
- Phát triển chức năng đăng ký và đăng nhập.
