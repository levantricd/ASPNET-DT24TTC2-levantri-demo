# Website đặt lịch hẹn khám bệnh ngoài giờ

## Giới thiệu
Đây là dự án xây dựng website đặt lịch hẹn khám bệnh ngoài giờ, giúp người bệnh dễ dàng đặt lịch khám trực tuyến.
## Mục tiêu
- Đăng ký tài khoản và đăng nhập.
- Đặt lịch khám trực tuyến.
- Quản lý thông tin bệnh nhân.
- Quản lý lịch khám.
- Tra cứu lịch hẹn và kết quả đặt lịch.

## Công nghệ sử dụng
- ASP.NET Core (.NET 10)
- C#
- Entity Framework Core
- SQL Server
- HTML, CSS, Bootstrap
- JavaScript

## Chức năng chính
- Đăng ký, đăng nhập người dùng.
- Quản lý thông tin bệnh nhân.
- Đặt lịch khám theo chuyên khoa hoặc bác sĩ.
- Quản lý lịch hẹn.
- Trang quản trị hệ thống.
### Quản trị viên

- Quản lý bác sĩ
- Quản lý lịch khám
- Quản lý thông tin đặt lịch
- Quản lý bệnh nhân
